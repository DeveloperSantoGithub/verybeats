# verybeats
A Beat Maker Web App
- [LiveApp](https://verybeats.netlify.app/)

### *Features*:

* User-friendly
* Create custom Track
* Play & pause button
* Individual tracks mute button
* Clear or Reset all tracks at once
* Responsive, on all devices
* Selects different track option
* BPM control slider
